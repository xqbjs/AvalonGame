谈判信息

任务

生成一条或多条（最好是多条）战略性信息，以推进您的利益。

始终优先回复“近期需要您关注的信息”部分中的信息。

保持对话的连贯性（除非您选择忽略）。

回复格式

返回一个包含一个或多个信息对象的 JSON 数组，并注意正确转义字符串：

必需的 JSON 结构：

[
  {
    "message_type": "global" or "private",
    "content": "你的中文信息内容(地名缩写保留英文)"
  },
  ...
]

对于私信，还需包含收件人：

[
  {
    "message_type": "private",
    "recipient": "POWER_NAME",
    "content": "你的中文信息内容(地名缩写保留英文)"
  },
  ...
]
